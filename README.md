### Release

CircleCI builds the plugin continuously.
It also publishes an artifact if a tag is pushed and following variables are set.

Environment Variable        | Value
----------------------------|------
`$BINTRAY_USER`             | Bintray user name
`$BINTRAY_KEY`              | Bintray key
